# Angular `Apple` Music List display

Angular Apple Music-Lists using Angular, RxJS and new Angular Material.


## Instruction and deployment:

â–  Setup a default seed project or use CLI:

- Install nodeJS & npm (or simply verify you already have it):
```javascript
  node -v
  npm -v
```  

- Install project
```javascript
  npm i
```  

- Install the Angular CLI globally (If not installed previouly):
```javascript
  npm install -g @angular/cli
  ng -v
```

- Serve the application:
```javascript
  cd angular-apple-music-lists
  npm run start
```
